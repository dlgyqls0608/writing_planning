# NovelForge — 화면 와이어프레임

> 버전: 1.0 | 기준 문서: Planning.md v1.0
>
> **표기 규칙**
> - `[ 버튼명 ]` = 클릭 버튼
> - `[____________]` = 텍스트 입력 필드
> - `☐ / ☑` = 체크박스 | `◉ / ○` = 라디오 버튼
> - `▼` = 드롭다운 | `▶` = 펼치기/접기
> - `▓▓▓` = 활성 탭 | `░░░` = 비활성 탭
> - `///` = 일러스트·이미지 자리표시자
> - `※` = UI 동작 설명 (화면에 표시 안 됨)

---

## 화면 흐름 (User Flow)

```
[S-01 작품 목록]
      │
      ▼ 작품 선택 또는 새 작품
[S-02 메인 화면 — 3패널]
      │
      ▼ 아이디어 입력 클릭
[S-03 아이디어 입력]
      │
      ▼ 제출
[S-04 AI 확인 질문]
      │
      ▼ 답변 완료
[S-05 문서 선택]
      │
      ├── 로그라인 ──── [S-06]
      ├── 시놉시스 ──── [S-07]
      ├── 플롯 ─────── [S-08]
      ├── 트리트먼트 ── [S-09]
      └── 스토리 바이블 [S-10]
                │
                ▼ (선택)
         [S-11 복선 트래커]
         [S-12 성공 패턴 체크리스트]
                │
                ▼
         [S-13 저장 및 내보내기]
```

## S-04 | AI 확인 질문

```
┌─────────────────────────────────────────────────────────────────────┐
│  NovelForge  /  새 작품  /  AI 확인 질문                [← 수정하기] │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  입력하신 아이디어                                                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "재벌 비서로 일하다가 억울하게 죽은 여자가 10년 전으로...   │    │
│  │  ...숨겨진 진범이 따로 있다는 걸 알게 되는 이야기"          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  방향을 조금 더 알려주시면 더 잘 정리할 수 있어요. (2~3개)            │
│  "모르겠다"를 선택하면 빈 칸으로 두고 나머지를 먼저 만들어 드립니다. │
│                                                                       │
│  ─────────────────────────────────────────────────────────────────  │
│  Q1. 이 이야기의 분위기는 어느 쪽에 가깝나요?                         │
│                                                                       │
│    ○  통쾌한 복수극 (적을 하나씩 무너뜨리는 카타르시스)               │
│    ○  서스펜스·스릴러 (진범을 찾아가는 긴장감 위주)                    │
│    ○  멜로 중심 (복수보다 감정·관계가 더 중요)                         │
│    ○  모르겠다                                                         │
│                                                                       │
│  ─────────────────────────────────────────────────────────────────  │
│  Q2. 주인공이 결국 복수에 성공하는 결말인가요?                         │
│                                                                       │
│    ○  성공 (완전한 복수 + 해피엔딩)                                    │
│    ○  성공하지만 대가를 치른다                                         │
│    ○  복수보다 더 중요한 것을 택한다                                   │
│    ○  모르겠다                                                         │
│                                                                       │
│  ─────────────────────────────────────────────────────────────────  │
│  Q3. 로맨스 상대가 있다면 누구인가요?                                  │
│                                                                       │
│    [___________________________________]  (예: 재벌 회장 아들, 없음)  │
│                                                                       │
│  ─────────────────────────────────────────────────────────────────  │
│                                    [     다음: 문서 선택 →      ]     │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```
※ 모든 항목에 "모르겠다" 선택 가능 → 해당 항목은 문서에서 빈 칸으로 처리  
※ `[다음]` → S-05 문서 선택으로 이동
